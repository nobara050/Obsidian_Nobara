## Base on / Based on / Basing on

-- S + base + object + on + noun

-- Based on + danh từ/cụm danh từ, S + V

-- Basing + object + on + something, S + V

## Never before

**-- S + have/has + never before + V3/V-ed**

I have never before seen such a beautiful place.
She has never before worked with foreign clients.

**-- Never before + have/has/had + S + V3/V-ed**
## V + off

The button has come off your jacket.
**come off** = **rời ra, bong ra, tuột ra khỏi một bề mặt/vật gì đó**

**pull off** = **kéo ra / giật ra / tháo ra**

**go off** thường có nghĩa:

The alarm went off.
= Chuông báo reo.

The bomb went off. 
= Quả bom phát nổ.

The milk has gone off. 
= Sữa đã hỏng.
